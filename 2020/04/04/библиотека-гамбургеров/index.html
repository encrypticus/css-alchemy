Библиотека гамбургеров

Вдохновившись этой небольшой CSS-библиотекой решил написать подобную, но со своими эффектами. Итак, встречайте - Magic burgers. Magic burgers - это небольшая css/sass-библиотека анимированных гамбургер-меню. Помимо css-файла библиотека включает исходные sass-файлы. Библиотека модульная и настраиваемая, что позволяет вам приготовить свой собственный гамбургер! Гамбургеры на изображении ниже кликабелны.

Для того, чтобы начать использовать библиотеку, установите её из реестра NPM. Затем, если вы используете препроцессор Sass, импортируйте в вашем файле главный файл библиотеки hamburgers.scss ( также доступна sass-версия ). Или же в секции <head > на странице вашего сайта вставьте следующий код:

<link href="path/to/hamburger.min.css" rel="stylesheet">

Затем на страницу вашего сайта добавьте следующую разметку:


<button class="hamburger">
  <span class="hamburger-inner">
    <span class="line-top">
      <span class="line-top-left"></span>
      <span class="line-top-right"></span>
    </span>
    <span class="line-center">
      <span class="line-center-left"></span>
      <span class="line-center-right"></span>
    </span>
    <span class="line-bottom">
      <span class="line-bottom-left"></span>
      <span class="line-bottom-right"></span>
    </span>
  </span>
</button>

Добавьте название класса того типа гамбургера, который вам нужен:


<button class="hamburger hamburger--arrowdown">
  <span class="hamburger-inner">
    <span class="line-top">
      <span class="line-top-left"></span>
      <span class="line-top-right"></span>
    </span>
    <span class="line-center">
      <span class="line-center-left"></span>
      <span class="line-center-right"></span>
    </span>
    <span class="line-bottom">
      <span class="line-bottom-left"></span>
      <span class="line-bottom-right"></span>
    </span>
  </span>
</button>

Доступны следующие классы:

hamburger--minify

Чтобы переключить состояние гамбургера, добавьте класс is-active:


<button class="hamburger hamburger--arrowdown is-active">
  <span class="hamburger-inner">
    <span class="line-top">
      <span class="line-top-left"></span>
      <span class="line-top-right"></span>
    </span>
    <span class="line-center">
      <span class="line-center-left"></span>
      <span class="line-center-right"></span>
    </span>
    <span class="line-bottom">
      <span class="line-bottom-left"></span>
      <span class="line-bottom-right"></span>
    </span>
  </span>
</button>

Поскольку имя класса должно переключаться с помощью JavaScript, в директории js/ находится файл hamburger.js, код в котором предназначен для этого. Просто подключите его на странице вашего сайта перед закрывающим тегом </body>, вставив следующий код:

<script src="js/hamburger.js"></script>

Также, если вы используете препроцессор pug, в директории pug/ находится файл hamburger.pug с примесью, создающей разметку для гамбургера. Подключив файл с примесью в своем коде, вы можете вызвать ее следующим образом:

+hamburger("class name")

Где class name - это название класса того типа гамбургера, который вам нужен. Например:

+hamburger("3d")
+hamburger("alchemy")
+hamburger("juggler")
+hamburger("shurikens-alt-r")

Sass

Исходные файлы .scss и .sass доступны, если вы используете препроцессор Sass. Это дает вам гибкость и модульность.

  • Импортируйте файл hamburger.scss внутри вашего scss-файла стилей:
    @import "path/to/hamburger";
  • Настройте свой гамбургер - Перед импортом главного файла библиотеки переопределите переменную $hamburger-types, чтобы удалить из скомпилированного файла те типы гамбургеров, которые вы не планируете использовать. Это позволит уменьшить размер скомпилированного css-файла:
$hamburger-types: ( alchemy );
@import "hamburger"

Кастомизация

Чтобы переопределить настройки по умолчанию, объявите их перед импортом гамбургеров:

$hamburger-layer-width: 30px;
$hamburger-layer-height: 3px;
$hamburger-layer-spacing: 5px;
@import "hamburger";

Вы также можете создать отдельный файл (например, hamburger-settings.scss) с этими настройками, а затем импортировать его перед гамбургерами:

@import "hamburger-settings";
@import "hamburger";

Вот полный список настроек по умолчанию (находится в sass/hamburger.scss):

$hamburger-layer-width        : 40px;
$hamburger-layer-height       : 4px;
$hamburger-layer-spacing      : 6px;
$hamburger-padding            : 10px;
$hamburger-layer-border-radius: 4px;
$hamburger-layer-color        : #000;
//Переопределите эту переменную, чтобы удалить из скомпилированного файла те типы гамбургеров,
//которые вы не планируете использовать
$hamburger-types: (
simple,
collapse,
storm,
juggler,
shurikens,
shurikens-r,
shurikens-alt,
shurikens-alt-r,
minify,
arrowturn,
arrowturn-r,
arrowdown,
arrowdown-r,
magic,
alchemy,
alchemy-r
);

Поддержка браузерами

Некоторые анимации используют CSSPerspective API, который на момент написания данной статьи не поддерживается в IE и Firefox. Для подробной информации воспользуйтесь caniuse.com.
